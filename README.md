# 🎬 MovieMate

A modern React-based movie search application that allows users to discover movies, view details, rate them, and maintain a personalized watched list.

---

## 🚀 Features

- 🔍 Search movies using IMDb API
- 🎥 View detailed movie information
- ⭐ Rate movies after watching
- 📂 Add/remove movies from a watched list
- ⚡ Fast and responsive UI
- 🧠 Built with reusable components

---

## 🛠️ Technologies Used

- **Frontend:** React.js
- **API:** IMDb (via OMDb API)
- **Styling:** CSS
- **State Management:** React Hooks

---

🪝 Custom Hooks

To keep the code clean and scalable, custom hooks were implemented:

🔑 useKey
Handles keyboard events
Used for actions like closing movie details (e.g., Escape key)
Improves user experience with keyboard shortcuts

🎬 useMovies
Handles fetching movie data from the API
Manages loading and error states
Keeps API logic separate from UI components

## 🧠 Concepts Learned

This project focuses on strengthening core React fundamentals:

- `useState` – Managing component state
- `useEffect` – Handling side effects like API calls
- Component-based architecture
- Breaking UI into reusable components
- Props and data flow

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/moviemate.git

# Navigate to project folder
cd moviemate

# Install dependencies
npm install

# Start the app
npm start
```

---

## ⚙️ How to Use

1. Search for any movie using the search bar
2. Click on a movie to view details
3. Rate the movie after watching
4. Add it to your watched list
5. Manage your watched movies anytime
